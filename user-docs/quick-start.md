# 🚀 快速开始

本指南将帮助您在 5 分钟内快速上手 InfloWave，体验其核心功能。

## 📋 准备工作

在开始之前，请确保：

1. **已安装 InfloWave** - 如未安装，请参考 [安装指南](./installation.md)
2. **有可用的 InfluxDB 服务器** - 可以是本地或远程的 InfluxDB 1.x 实例
3. **知道连接信息** - 服务器地址、端口、用户名和密码

> 💡 **提示**: 如果您还没有 InfluxDB 服务器，可以参考 InfluxDB 官方文档快速搭建一个测试环境。

## 🎯 第一步：启动应用

1. 双击桌面图标或从开始菜单启动 InfloWave
2. 首次启动会看到欢迎界面和空的连接列表
3. 界面分为左侧连接面板和右侧主工作区

## 🔗 第二步：添加数据库连接

### 创建新连接

1. 点击左上角的 **"+ 添加连接"** 按钮
2. 在弹出的对话框中填写连接信息：

   ```
   连接名称: 我的测试数据库
   服务器地址: localhost (或您的服务器IP)
   端口: 8086
   用户名: admin (或您的用户名)
   密码: ******
   ```

3. 点击 **"测试连接"** 验证配置是否正确
4. 测试成功后，点击 **"保存"** 创建连接

### 连接状态

- 🟢 **绿色圆点** - 连接正常
- 🔴 **红色圆点** - 连接失败
- 🟡 **黄色圆点** - 连接中

## 🗄️ 第三步：浏览数据库

### 查看数据库列表

1. 在左侧连接面板中点击您刚创建的连接
2. 右侧会显示该连接下的所有数据库
3. 如果没有数据库，可以点击 **"创建数据库"** 创建一个

### 创建测试数据库

1. 点击 **"创建数据库"** 按钮
2. 输入数据库名称，例如：`test_db`
3. 点击 **"确认"** 创建

### 浏览数据库内容

1. 点击数据库名称展开详情
2. 可以看到：
   - **保留策略** - 数据保留时间设置
   - **测量列表** - 类似于关系数据库中的表
   - **字段和标签** - 数据结构信息

## 🔍 第四步：执行查询

### 切换到查询页面

1. 点击顶部导航栏的 **"查询"** 标签
2. 或使用快捷键 `Ctrl+2` (Windows/Linux) 或 `Cmd+2` (macOS)

### 编写第一个查询

1. 在查询编辑器中输入：
   ```sql
   SHOW DATABASES
   ```

2. 点击 **"执行"** 按钮或按 `Ctrl+Enter`
3. 查询结果会显示在下方的结果面板中

### 查询数据

如果您的数据库中有数据，可以尝试：

```sql
-- 查看所有测量
SHOW MEASUREMENTS

-- 查询最近的数据
SELECT * FROM "your_measurement" LIMIT 10

-- 按时间范围查询
SELECT * FROM "your_measurement" 
WHERE time > now() - 1h
```

### 查询功能特色

- **语法高亮** - InfluxQL 语法自动高亮
- **智能提示** - 数据库、测量、字段名自动补全
- **查询历史** - 所有查询自动保存，可随时查看
- **结果导出** - 支持导出为 CSV、JSON、Excel 格式

## 📊 第五步：创建可视化图表

### 切换到可视化页面

1. 点击顶部导航栏的 **"可视化"** 标签
2. 或使用快捷键 `Ctrl+3`

### 创建第一个图表

1. 点击 **"新建图表"** 按钮
2. 选择图表类型（推荐从折线图开始）
3. 配置数据源：
   - 选择连接和数据库
   - 输入查询语句
   - 设置时间字段和数值字段

4. 点击 **"预览"** 查看图表效果
5. 满意后点击 **"保存"** 保存图表

### 图表类型

- **折线图** - 适合时序数据趋势展示
- **柱状图** - 适合分类数据对比
- **饼图** - 适合比例数据展示
- **面积图** - 适合累积数据展示

## 📥 第六步：导入数据（可选）

### 切换到数据写入页面

1. 点击顶部导航栏的 **"数据写入"** 标签
2. 或使用快捷键 `Ctrl+4`

### 导入 CSV 文件

1. 点击 **"导入文件"** 按钮
2. 选择您的 CSV 文件
3. 配置字段映射：
   - 时间字段
   - 测量名称
   - 标签字段
   - 数值字段

4. 预览数据确认无误
5. 点击 **"开始导入"** 执行导入

## 💡 实用技巧

### 右键快捷操作

在数据库和测量列表中右键点击，可以快速执行：
- 查看数据
- 复制名称
- 刷新列表
- 删除项目

### 键盘快捷键

- `Ctrl+1` - 切换到仪表板
- `Ctrl+2` - 切换到查询页面
- `Ctrl+3` - 切换到可视化页面
- `Ctrl+4` - 切换到数据写入页面
- `Ctrl+Enter` - 执行查询
- `Ctrl+S` - 保存查询

### 查询书签

1. 在查询编辑器中编写查询
2. 点击 **"保存为书签"** 按钮
3. 输入书签名称和描述
4. 下次可以从书签列表快速加载
