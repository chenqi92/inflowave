## 🚀 v1.0.0 (首个正式版本)

**发布日期**: 2023-08-01

### ✨ 核心功能
- **数据库连接** - 支持 InfluxDB 1.x 连接和认证
- **数据库管理** - 数据库创建、删除、查看功能
- **查询执行** - 基于 Monaco Editor 的 InfluxQL 查询编辑器
- **数据可视化** - 基础的折线图和柱状图支持
- **数据写入** - 支持 Line Protocol 格式的数据写入
- **文件导入** - 基础的 CSV 文件导入功能

### 🏗️ 技术架构
- **前端**: React 18 + TypeScript + Ant Design
- **后端**: Tauri 2.0 + Rust
- **图表**: ECharts 5.x
- **编辑器**: Monaco Editor
- **状态管理**: Zustand

### 🎯 平台支持
- **Windows**: x64, x86
- **macOS**: Intel (x64), Apple Silicon (ARM64)
- **Linux**: x64, ARM64, x86
