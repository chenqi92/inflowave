# InfloWave v0.8.4 发布说明

## 🚀 数据表格交互体验全面升级

本版本专注于数据表格的交互体验优化，带来了类 Excel 的列筛选、三态排序、列重排序、智能单元格选择等一系列强大功能，大幅提升数据浏览和操作效率。

---

## ✨ 重要特性

### 📊 类 Excel 表格功能

#### 列筛选与排序
- **新增** 类 Excel 的列筛选功能，支持快速过滤数据
- **实现** 三态排序系统（升序 → 降序 → 无序）
- **优化** 筛选图标位置计算，确保精准定位
- **改进** 筛选弹出框定位逻辑，避免遮挡内容
- **重构** 列标题排序箭头显示方式
- **优化** 筛选功能和排序图标样式

#### 列操作增强
- **新增** 列重排序功能，支持拖拽调整列顺序
- **移除** 列宽的 localStorage 持久化，避免缓存问题
- **优化** 列宽自适应计算逻辑

### 🎯 单元格选择系统

#### 可视化选择
- **新增** 单元格、行、列的可视化选择边框
- **添加** 选择区域的内部分隔线
- **增强** 选择边框渲染效果
- **优化** 选择边框定位和 z-index 层级
- **修复** 表头选择背景的 1px 偏移问题

#### 交互优化
- **新增** 拖拽选择时的自动滚动功能
- **增强** 单元格选择交互体验
- **处理** 无效网格列的边界情况
- **调整** 选择边框的 z-index 值

### 🖱️ 光标与交互

#### 光标处理
- **增强** Glide Data Grid 的光标处理
- **改进** 桌面应用中的光标样式
- **修复** 光标样式在不同场景下的显示问题
- **优化** 数据表格中的光标处理逻辑

### 🎨 UI/UX 优化

#### 表格渲染
- **优化** Glide Data Grid 渲染逻辑
- **调整** 容器尺寸和滚动行为
- **修复** 表格边框缺失问题
- **添加** 行标记宽度配置
- **调整** CSS z-index 值

### 🗄️ 数据库浏览器

- **优化** MultiConnectionTreeView 中的节点展开逻辑
- **改进** 树形视图的性能和稳定性
